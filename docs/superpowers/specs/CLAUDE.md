- `2026-04-12-step-prompt-design.md` — per-step `prompt:` field in workflow definitions
- `2026-04-13-trace-logging-design.md` — trace logging + replay design
- `2026-04-14-structured-outputs-design.md` — replacing substring control flow with the `submit_result` tool
